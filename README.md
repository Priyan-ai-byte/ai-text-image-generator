# AI Text & Image Generator

This project is a simple AI tool that can generate text and images from user prompts.

## Features
- Text Generation using a language model
- Image Generation using Stable Diffusion

## Technologies Used
- Python
- PyTorch
- Hugging Face Transformers
- Diffusers (Stable Diffusion)

## Project Structure

ai-text-image-tool
│
├── models
│   ├── text_model.py
│   └── image_model.py
│
├── outputs
│
├── main.py
├── requirements.txt
└── README.md

## Installation

Clone the repository and install dependencies:

pip install -r requirements.txt

## Run the Program

python main.py

Then choose:
1 → Text generation  
2 → Image generation
